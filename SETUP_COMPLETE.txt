DATABASE SCHEMA
═════════════════════════════════════════════════════════════════════════════

Companies (1)
  ├── Trucks (N) - health_score, capacity_tons, mileage_km
  ├── Drivers (N) - years_experience, accidents_count
  └── Bilties (N) - sender/receiver details, weight, rate
      └── Bilty → Route → distance_km, duration
      └── Bilty → Trip (N) - real-time tracking
          ├── Trip → Truck
          ├── Trip → Driver
          └── Trip → CascadeRisk (N) - network dependencies

MLPrediction - Historical predictions for model evaluation

QUICK START
═════════════════════════════════════════════════════════════════════════════

### Windows
    .\setup.bat

### Linux / macOS
    bash setup.sh

### Manual Setup

Backend:
    cd backend
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    python main.py

Frontend:
    cd frontend
    npm install
    npm run dev

### Docker
    docker-compose up

Then visit:
    • Frontend: http://localhost:5173
    • API: http://localhost:8000
    • Docs: http://localhost:8000/docs

PRODUCTION DEPLOYMENT
═════════════════════════════════════════════════════════════════════════════

Docker:
    docker build -t biltybook:latest backend/
    docker run -p 8000:8000 biltybook:latest

Docker Compose:
    docker-compose up

Environment Variables:
    DATABASE_URL=postgresql://user:pass@host:5432/biltybook
    GEMINI_API_KEY=your_api_key_here
    API_HOST=0.0.0.0
    API_PORT=8000

Pre-deployment Checklist:
    □ Database password updated
    □ HTTPS/SSL configured
    □ JWT authentication enabled
    □ CORS properly restricted
    □ Error logging configured
    □ Database backups scheduled
    □ Monitoring/alerting setup
    □ Load testing completed
    □ Security audit passed
